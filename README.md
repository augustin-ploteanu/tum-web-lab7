# 🎬 MyMovieList

A personal movie and TV series tracker. Search for titles, organize them into categories, track episodes, grade entries, and keep notes, all saved locally in your browser.

[Live Demo]()

---

## Features

- **Search** movies and TV shows via the TMDB API
- **My List** — add entries to one of four categories: Watching, Completed, Plan to Watch, Dropped
- **Episode tracking** — track watched vs total episodes
- **Grade & notes** — rate entries 1–10 and attach a personal note
- **Stats panel** — per-category counts, total episodes watched, and average grade
- **Search, sort & filter** within the list 
- **Grid / list view**
- **Light / dark theme**
